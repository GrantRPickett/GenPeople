Welcome to NPC Agency. A program designed to provide minor characters for video games. Currently used with the Inform engine. (http://inform7.com/download/) To create a completely random character just push the blue button. If you would like to adjusts some aspects of the character, expand the accordions below and make some changes before submitting the form. There are buttons below to download the Inform extension (to be installed by the user). A JSON serialization of the character can be loaded into this form for modification and changes saved. The basic information is the most straight forward and has details about where the person was born, how old he is, etc. The traits affect which events a person might experience and their description. The Eras section lets you influence the events in each era to be more positive or negative.
